⚙️ API Configuration

Configure your WhatsApp Cloud API credentials. These are stored in the .env file on the server.

⚠️ Edit the .env file on the server to change these settings. Restart the server after changes.

📘 Quick Setup Guide

  1. Create a Meta Developer Account at developers.facebook.com
  2. Create a new Business App and add the WhatsApp product
  3. Go to WhatsApp → API Setup → Copy your Phone Number ID and Access Token
  4. Paste them in your .env file
  5. Create Message Templates in Meta Business Suite and wait for approval
  6. Set up the Webhook URL (use ngrok for development)
  7. Upload contacts and start sending campaigns! 🎉
